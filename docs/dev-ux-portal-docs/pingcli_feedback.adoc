Help us improve the CLI. Report issues or send us feedback on using the CLI tool.

== Synopsis

Provides links to report issues and provide feedback on using the CLI to Ping Identity.

----
pingcli feedback [flags]
----

== Examples

----
  pingcli feedback
----
