List all custom configuration profiles.

== Synopsis

List all custom configuration profiles stored in the CLI configuration file.

----
pingcli config list-profiles
----

== Examples

----
  List all custom configuration profiles stored in the CLI configuration file.
    pingcli config list-profiles
----
